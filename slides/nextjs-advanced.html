Next.js 고급 기능

React 기반 풀스택 프레임워크

Next.js란?

  • React 기반 풀스택 프레임워크
  • 서버 사이드 렌더링 (SSR) 지원
  • 정적 사이트 생성 (SSG) 지원
  • 자동 코드 분할
  • API 라우트 내장

라우팅 시스템

  • 파일 기반 라우팅: 폴더 구조가 URL 구조
  • 동적 라우팅: [id] 폴더로 동적 경로
  • 중첩 라우팅: 폴더 중첩으로 중첩 레이아웃
  • 라우트 그룹: (group) 폴더로 논리적 그룹화

데이터 페칭

// 서버 컴포넌트
async function getData() {
  const res = await fetch('https://api.example.com/data');
  return res.json();
}

export default async function Page() {
  const data = await getData();
  return <main>{data.title}</main>;
}

렌더링 방식

  • 서버 컴포넌트: 서버에서 렌더링
  • 클라이언트 컴포넌트: 브라우저에서 렌더링
  • 정적 생성: 빌드 시 HTML 생성
  • 동적 렌더링: 요청 시 렌더링

최적화 기능

  • 이미지 최적화: next/image 컴포넌트
  • 폰트 최적화: next/font
  • 자동 코드 분할: 페이지별 분할
  • 캐싱: 자동 캐싱 전략
